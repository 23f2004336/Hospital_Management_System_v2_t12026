Setup & Installation

⚠️ You need 5 terminals running simultaneously. Keep them all open throughout development.

Prerequisites

Python 3.8+
python --version
Node.js 14+
node --version
Redis Server
Ubuntu: sudo apt install redis-server
Mac: brew install redis
npm 6+
npm --version
Terminal 1 — Redis Server

# Start Redis — must be running before anything else

$ redis-server

 

# Verify it works

$ redis-cli ping

PONG ← you should see this

Terminal 2 — Flask Backend

$ cd backend

# Create virtual environment

$ python -m venv venv

 

# Activate — Linux/Mac

$ source venv/bin/activate

# Activate — Windows

$ venv\Scripts\activate

 

$ pip install -r requirements.txt

$ python app.py

 

Admin Created Successfully

* Running on http://127.0.0.1:5000

Terminal 3 — Celery Worker

$ cd backend && source venv/bin/activate

$ celery -A app.celery worker --loglevel=info

# Handles background tasks: CSV export, emails

Terminal 4 — Celery Beat Scheduler

$ cd backend && source venv/bin/activate

$ celery -A app.celery beat --loglevel=info

# Triggers: daily reminders (8AM) + monthly reports (1st of month)

Terminal 5 — Vue.js Frontend

$ cd frontend

$ npm install

$ npm run serve

 

App running at: http://localhost:8080

✅ Open http://localhost:8080 in your browser. All 5 terminals must remain running.

Roles & Features

👑
Admin
Hospital Staff — Full system access
Pre-created · No registration
  • Dashboard with real-time stats: total doctors, patients, appointments
  • Add, update, and delete doctor profiles (name, specialization, fee, qualification)
  • View all hospital appointments — upcoming and past
  • Search patients and doctors by name or specialization (client-side filtering)
  • Remove/blacklist patients from the system with cascade delete
  • Manually trigger daily reminder and monthly report jobs from the dashboard
🩺
Doctor
Medical professional registered by Admin
Created by Admin only
  • Dashboard with upcoming (Booked/Confirmed) and past appointments
  • Set 7-day availability schedule with custom start/end times per day and off-duty days
  • Confirm or cancel patient appointments
  • Add treatment records: diagnosis, prescription, notes, next visit date
  • View full medical history of any assigned patient
  • Edit own treatment records only — cannot modify other doctors' records (security enforced)
🧑‍⚕️
Patient
End user seeking medical care
Self-registration allowed
  • Self-register and login with secure hashed password
  • Search doctors by name or specialization with dropdown filter
  • View doctor availability for next 7 days before booking
  • Book appointments with conflict prevention (no double-booking same slot)
  • Reschedule or cancel existing appointments
  • View full appointment history with diagnosis, prescription, notes, and next visit
  • Export complete treatment history as CSV (async job — sent via email)
  • Edit personal profile: name, gender, DOB, address, phone
